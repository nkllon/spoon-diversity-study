# Design: Spoon Ontology v0.3

- Namespaces: sp, prov, sosa, ssn, schema, xsd, rdfs, owl
- Classes: Participant, Tool, ToolSet, Task, Trial (prov:Activity), ToolUseObservation (sosa:Observation)
- Object props: associatedParticipant, usesToolSet, informedByTask, observesTrial
- Datatype props: handSizeCm, timeSeconds, numOperations, errorRatePercent, switchCount, switchTimeSeconds, perceivedCognitiveCost, perceivedDiscomfort
- Controlled: define `sp:Task` and `sp:ToolSet` individuals
