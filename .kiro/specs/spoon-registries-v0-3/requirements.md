# Requirements: Spoon Registries v0.3

## 1. Spore Registry
THE system SHALL track spores and link to key artifacts with PROV.

## 2. Guidance Registry
THE system MAY track plan-of-day references to documentation.

## 3. URI & Fields
THE registries SHALL use dated path base IRIs and include label, createdAt/lastUpdated, status, and PROV links.
